# ⚡ Meeting Brief Generator

> Walk into every meeting prepared. Paste a meeting title or invite → get a sharp, structured 1-page brief in seconds.

Built at the **Manus x SXSW Sydney Hackathon** in under 35 minutes.

![Status](https://img.shields.io/badge/status-live-brightgreen) ![Built with](https://img.shields.io/badge/built%20with-Claude%20API-orange)

---

## What it does

Paste any meeting title or calendar invite text and instantly get:

- 🎯 **Meeting Goal** — the single most important outcome
- 👥 **Key Attendees & Their Agenda** — who's in the room and what they want
- ❓ **Smart Questions to Ask** — specific, not generic
- 📚 **Background Context** — dynamics and norms to be aware of
- 💬 **Your Talking Points** — concrete positions to walk in with

---

## Tech Stack

- **Frontend** — Vanilla HTML/CSS/JS (single file, zero dependencies)
- **AI** — Anthropic Claude API (`claude-sonnet-4-20250514`)
- **Hosting** — GitHub Pages

---

## How to run locally

Just open the file in your browser — no server needed.

```bash
open meeting-brief-generator.html
```

That's it. It calls the Anthropic API directly from the browser.
